Meet the Team

Project Manager & Team Lead

Ololade

Coordinates the entire project lifecycle, focusing on final written deliverables (Technical Poster, Presentation Slides, Individual Reflection tracking), and overall time management.

Lead Device Programmer (C++)

Ali

Writes the C++ code for the ESP32: handles sensor reading, implements alert logic (LEDs/buzzer code update), manages the I2C LCD display, and is responsible for the Final Code Submission.

Hardware & Wiring Specialist

Favour

Leads the physical Arduino Hardware Set Up. Builds and wires the ESP32, DHT22 sensor, and alert components (LED & Buzzer hardware setup) on the breadboard.

Dashboard & Viz Specialist

Bless

Specializes in Dashboard Design & Development. Builds the Node-RED interface, subscribing to MQTT data to create visual components like gauges and real-time charts.

Cloud & Media Engineer

Alex

Responsible for MQTT Configuration (including C++ code for data publishing and broker setup) and recording the Video Demonstration (Nov 24th milestone).
